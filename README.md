# Retail Sales Performance & Customer Insights Analysis

**Author:** Omkar Shirsath
**Role:** Aspiring Data Analyst

---

## Business Objective

This project aims to analyze retail sales data to track performance, measure key business metrics, and uncover customer purchasing patterns.
The goal is to support decision-making in areas such as product strategy, inventory planning, and revenue improvement.

---

## Dataset (To Be Added)

A retail or e-commerce transactional dataset will be used.
It is expected to include information such as:

* Orders
* Products
* Customers
* Prices and quantities
* Dates
* Locations

The dataset will be uploaded in the `data/` folder once finalized.

---

## Tools Planned

* SQL for data cleaning and metric calculation
* Excel / Power BI / Tableau for visualization and dashboards

---

## Planned Workflow

1. Understand dataset structure and relationships
2. Clean and validate the data
3. Create business KPIs
4. Analyze sales trends
5. Evaluate product and category performance
6. Study customer behavior
7. Build an interactive dashboard
8. Deliver insights and recommendations

---

* Category Contribution
* Repeat Customer Rate
